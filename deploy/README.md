# QuotePilot Deployment to Alibaba Cloud Function Compute

## Prerequisites

1. **Alibaba Cloud International Account**: You need an active account with access to international regions.
2. **Function Compute Activated**: Ensure FC is activated in the `ap-southeast-1` (Singapore) region.
3. **RAM User Permissions**: Create a RAM user with `AliyunFCFullAccess` policy attached.
4. **Access Key Pair**: Obtain an AccessKey ID and Secret for the RAM user.
